# First Genesis — Company Context

## What We Do
First Genesis is a boutique technology consultancy delivering AI-native digital
transformation for enterprise clients. Core offerings:
- Custom software (SaaS, platforms, mobile)
- AI/ML integration and data strategy
- 3D and spatial computing (silhouette, mesh design)
- Project portfolio management

## How We Deliver
1. **Charter** — PM Agent generates; CDO approves
2. **Requirements** — BA Agent extracts from design sessions; PM approves
3. **Build** — engineering team + vendors
4. **QA** — QA Agent audits before customer handoff
5. **Delivery** — Manager Agent tracks; CEO approves final release

## Current Focus (MVP Phase)
Four delivery agents are live: PM, BA, QA, Vendor.
Manager and all sales agents are idle until post-MVP.

## Client Interaction Model
- Design sessions via Zoom (transcripts fed to BA Agent)
- Deliverable approval via email gates
- Weekly vendor scorecards to leadership
- Monthly portfolio dashboard to Pascal (CEO)

## Quality Standards
- Zero undocumented scope changes
- All acceptance criteria must be met before QA passes
- Vendor SLA compliance reviewed weekly
- Every agent output logged to immutable audit trail
